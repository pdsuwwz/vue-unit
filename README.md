Unit Test for Vue 3.x + Vite 4.x


> For Typescript: https://github.com/pdsuwwz/vue-unit-ts

## Environment Support

* Jest 29.x
* Vue 3.x
* Node >= 14.19.x

## Install

Install node dependencies in all packages

```bash
pnpm install
```

## Run

Local Development

```bash
pnpm dev
```

## Test

Unit Testing

```bash
pnpm test
```

Test code coverage

```bash
pnpm test:coverage
```
